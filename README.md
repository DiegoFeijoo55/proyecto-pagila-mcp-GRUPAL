# Proyecto Pagila MCP - Python

Este proyecto implementa un servidor MCP en Python que permite la interacción entre un modelo LLM y la base de datos **Pagila** mediante consultas en lenguaje natural.

El objetivo es demostrar cómo un LLM puede consultar una base de datos relacional sin conexión directa, utilizando un **MCP Server** como intermediario seguro y estructurado.

## Tecnologías utilizadas

- Python
- MCP (Model Context Protocol)
- PostgreSQL
- Base de datos Pagila

## Estructura del proyecto

- Servidor MCP para gestionar las solicitudes del LLM
- Conexión a la base de datos Pagila
- Procesamiento de consultas en lenguaje natural

## Propósito académico

Este proyecto forma parte de una práctica académica orientada a comprender:

- La arquitectura MCP
- Demostrar por qué es mejor usar MCP en lugar de conexión directa a la base de datos
- El flujo completo desde la pregunta del usuario hasta la consulta SQL ejecutada

## Cómo ejecutar

1. Crear y activar el entorno virtual
2. Instalar dependencias
3. Ejecutar el servidor MCP
